Welcome to Raspberry Pi College

This is a biometric login system online quiz portal that is capable of making a user database and hosting online question rounds whose results are then used to generate real time leaderboards. Questions and other internal data can only be updated by the admins.

Functionality

  • Biometric logins from different locations where system is set up
  • Easy sign-up and login
  • Different modules with questions
